# Exam Portal Frontend

React-based frontend application for the Exam Portal system.

## Installation

```bash
npm install
```

## Running the Application

```bash
npm start
```

The application will open at http://localhost:3000

## Features

- User Authentication (Login/Signup)
- Admin Panel (Create, Update, Delete exams)
- Exam Browsing
- Exam Taking Interface
- Results Display
- User Dashboard
